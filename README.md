# タスク管理 for SpringBoot3

## 概要
* 専門学校の授業で作成したプログラムになります
* 仕様に関しては授業で定義した範囲になります
* IDEはEclipse STSを利用しています

## 実行
### DockerのMySQLを起動
~~~sh
$ docker compose up
~~~

### Eclipseへのimport
1. Eclipseにjavaフォルダをimportする
  * プロジェクトはgradeで構成されているのでgradeプロジェクトとしてimportする
2. Eclipse上にてSpringBootのアプリケーションを起動する
